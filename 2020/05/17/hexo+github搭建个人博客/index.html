Nodejs 下载安装

  • 下载地址

Git 下载安装

  • 下载地址

1. git配置用户名和邮件

  • 桌面右键 -> Git Bash Here

2. 在github上配置 ssh key

  • 创建一个仓库：new repository，命名为 kris-yangjs.github.io 注意：这里最好是用这种格式(用户名.github.io)避免之后出现一些小问题
  • 头像下拉 -> settings -> SSH and GPG keys -> New SSH Key -> Title = blog
  • 回到 Bash 中输入以下命令生成 public key
  • 用记事本打开 id_rsa.pub，复制其中的内容粘贴到 Key 框中，然后点击 Add SSH key 即可保存

3. 确认 key 是否能用（一般没问题）

4. 安装 Hexo 博客框架

5. 配置 Hexo

  • 创建一个文件夹作为 blog 的工作目录，我这里就命名为 blog
  • 关闭 Bash 窗口，到 blog 目录下右键 -> Git Bash Here
